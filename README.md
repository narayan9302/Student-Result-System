# 🎓 Student Result Management System (Java)

A console-based application built using Java to manage student records efficiently using OOP and file handling.

---

## 🚀 Features

- Add student records
- Automatic grade calculation
- View all student results
- Search student by roll number
- Persistent file storage

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- File Handling (BufferedReader, FileWriter)

---

## 📂 Project Structure

Student-Result-System-Java/
│── Main.java
│── results.txt
│── README.md

---

## ⚙️ How It Works

- Data is stored in `results.txt`
- Format:

Roll|Name|Marks|Grade

---

## ▶️ How to Run

### Compile
javac Main.java

### Run
java Main
